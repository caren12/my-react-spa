Agency Flow

Agency Flow is a simple React application that allows users to add, view, search, and delete projects. It is designed as a basic portfolio management tool for showcasing creative work.

Features
Add new projects with a title and description
Search projects by title
Delete projects from the list
Responsive layout for mobile and desktop
Built using reusable React components
Tech Stack
React (Vite)
JavaScript
HTML
CSS

How to Run the Project 
1. Install dependencies:
   npm install
2. Start the development server:
   npm run dev
